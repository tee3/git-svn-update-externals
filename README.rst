``git-svn-update-externals``
============================

Overview
--------

This is a script to enhance ``git-svn`` to honor ``svn:externals``
using Git submodules.

This may work with earlier versions of Git and ``git-svn``, but has
not been tested.

Requirements
------------

* Python 2.6
* Git 2.8.0
* ``git-svn`` 2.8.0
